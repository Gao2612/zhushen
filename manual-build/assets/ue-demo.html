Scope

本阶段只验证环境

不做战斗、不做技能、不做 Boss、不做正式 UI、不做正式角色美术、不做复杂特效、不做资料馆联动。v0.1 只回答一个问题：这个 UE5 项目能不能稳定跑起来并打成包。

Project

工程骨架

仓库内保存 UE5 工程骨架：ue5/ZhushenActionDemo。G 盘工作目录同步到 G:\ZhushenUE5Demo，方便后续在 Unreal Editor 中打开。

Windows

Windows 打包目标

Shipping、Win64、Pak/IoStore、关闭 Debug Files，输出目录建议为 Builds/Windows/v0.1/。

Android

Android 横屏目标

包名 com.zhushen.demo，应用名“诸神终应知晓 Demo”，横屏运行，Texture Format 优先 ASTC，v0.1 可先将 Game Data 放入 APK。
