## 地图内容

`L_v01_Startup` 由 `Tools/CreateV01StartupMap.py` 生成，包含：

- `V01_GreyboxGround_40m`：40m x 40m 灰盒地面。
- `V01_VisiblePlaceholder_Body`：圆柱体占位身体。
- `V01_VisiblePlaceholder_Head`：球体占位头部。
- `V01_GreyboxScale_North/East/South/West`：四个尺度标记方块。
- `V01_DirectionalLight`：基础方向光。
- `V01_SkyLight`：基础天空光。
- `V01_OverviewCamera`：俯视 45 度附近的场景观察相机。
- `V01_TitleText`：`Zhushen UE5 Demo v0.1` 文字标识。

说明：上一版只有地面和 PlayerStart，PlayerStart 在视图里不等同于可见角色，因此这次改为显式生成圆柱 + 球体占位角色。

## 环境问题

### Win64 SDK

UBT 多次输出：

```text
##PlatformValidate: Win64 INVALID 10.0.22621.0
```

因此当前机器还不能直接从正式 C++ 工程完整 `-build`。Windows SDK 安装器曾尝试补装，但下载 cab 过程中超时。后续需要安装完整 Windows 10/11 SDK 与 MSVC C++ 工具链后，再恢复 C++ target 正式编译。

### Android 设备

当前 adb 输出：

```text
List of devices attached
```

没有在线设备，因此 APK 安装、横屏、真机启动时间、帧率和发热无法验证。
