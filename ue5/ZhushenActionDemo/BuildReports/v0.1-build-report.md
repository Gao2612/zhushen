## 验收步骤

1. 运行 `Tools\check-v01-env.ps1`，确认 UE_ROOT、UnrealEditor、RunUAT、VS、MSBuild、Windows SDK、JDK、adb、Android SDK 和 NDK 状态。
2. 用 Unreal Editor 打开 `ZhushenActionDemo.uproject`。
3. 运行 `Tools\CreateV01StartupMap.py`，生成 `L_v01_Startup`。
4. 在编辑器 Play 模式确认地面、占位角色、相机和 HUD 可见。
5. 执行 Windows 打包脚本并填写 Windows 表格。
6. 执行 Android 打包脚本并填写 Android 表格。
7. 将实测包体、启动时间、平均帧率、发热和问题记录写回本报告。
